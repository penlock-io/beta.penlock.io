penlock

State-of-the-art seed phrase backup.

Penlock is an open-source, cryptographically secure, printable paper-computer. It guides users through secret-splitting their BIP39 seed phrase without an electronic device.

Note from the developer

This is a beta release, exclusively intended for peer-reviewing and early testing. The public release is planned for Q3 2024 and it is likely, but not guaranteed, to be backward-compatible with the present documents. If we break compatibility, this beta-version will still be available at the current address (beta.penlock.io).

Please note that the following pieces are still missing:

  • Illustrated examples
  • The password generation guide

Browser compatibility is still being worked on; Safari has known printing issues, while Firefox and Chromium-based browsers are well supported.

Getting things done

Split a Seed Phrase (2-of-3)

Difficulty: Medium

Split a seed phrase into 3 cryptographically secure "shares"; any 2 of these can be used to recover the seed phrase. This is the recommended backup strategy.

Time required:
  • 12 words: ~2 hours
  • 24 words: ~3 hours

You need to print:

  • The 2-of-3 Guide
  • A 2-of-3 Worksheet:
    12-word 18-word 24-word
  • The 2-of-M Wheel
  • Penlock Tiles
+Craft Knife +Scissors +Pencil +Eraser

Generate a Seed Phrase

Difficulty: Easy

Generate a 12-word seed phrase using real-world, provable randomness.

Time required: ~30 minutes

You need to print:

  • The Seed Phrase Guide
  • The Seed Phrase Worksheet
  • The Seed Phrase Wordlist
  • Penlock Tiles
+Wallet +Scissors +Pencil +Eraser
